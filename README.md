# HashV6 — TRX 奇偶预测投注平台

## 项目概述

基于 TRON 区块链的奇偶预测投注平台，运行于 mac 笔记本本地，支持 **几万人同时下单** 的并发场景。

- **桌面端**：Tauri（Rust 后端 + Vue 前端，打包后一键启动，无需安装环境）
- **后端语言**：Rust
- **目标平台**：macOS / Windows

---

## 核心玩法

### 单/双判定规则

用户投注后，取 **投注金额的最后一位数字** 判断单双：

| 示例金额   | 取值         | 结果 |
| ---------- | ------------ | ---- |
| 199.09 TRX | 最后一位 `9` | 单   |
| 184.98 TRX | 最后一位 `8` | 双   |
| 10.00 TRX  | 最后一位 `0` | 双   |

### 赔付规则

- **中奖**：赔付 **1.95 倍**（含本金）。投 10 TRX → 中奖后返回 19.5 TRX
- **未中**：不返还任何金额
- **大额不赔**：单笔投注金额超过 **阈值** 时，不参与赔付（配置项：`max_payout_amount`）

### 计算方式

仅一种模式：**20 区块**

- 每 20 个区块结算一次
- 区块号必须能被 20 整除（如 82471940、82471960）
- 用户投注后，在下个整数 20 区块计算结果

## 技术架构

### 整体架构：Tauri 桌面应用

```
┌─────────────────────────────────────────┐
│              Tauri 桌面端                 │
│  ┌─────────────┐    ┌─────────────────┐ │
│  │  Vue 前端    │    │   Rust 后端      │ │
│  │  (WebView)  │◄──►│   (核心逻辑)     │ │
│  └─────────────┘    └────────┬────────┘ │
│                              │          │
│                     ┌────────▼────────┐  │
│                     │   TronGrid      │  │
│                     │   RPC / WebSocket│  │
│                     └─────────────────┘  │
└─────────────────────────────────────────┘
```

**为什么选 Tauri：**

- 后端已是 Rust，前后端语言统一，集成成本低
- 打包后单个可执行文件，一键启动，无需安装 Node.js / Java 等环境
- 内存占用远低于 Electron（~10MB vs ~150MB）
- 支持 mac + Windows 同一套代码
